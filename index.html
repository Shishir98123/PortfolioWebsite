About Me.

Hello! I'm Shishir Khatiwada, a web developer proficient in HTML, CSS, and JavaScript. I create interactive and visually appealing websites that leave a lasting impression.

My journey into web development started with a passion for creativity and problem-solving. With strong front-end skills, I craft engaging user experiences and responsive designs.

I'm enthusiastic about staying up-to-date with web technologies and expanding my skill set. I'm excited to collaborate on exciting projects and bring ideas to life.

  • My Skills Are: HTML, CSS, JavaScript, Git & Github, Figma & Tailwind.

Services.

Web Development

Using my expertise in HTML, CSS, and JavaScript, I have built responsive and user-friendly websites for various clients. Whether it's a simple landing page or a complex web application, my goal is to provide functional and visually appealing web solutions. I also leverage tools like Git/GitHub for version control and Tailwind CSS for efficient styling, ensuring a seamless development process.

User Interface (UI) Design

I employ Figma to create wireframes and high-fidelity prototypes that showcase the user experience. Collaborating closely with developers, I provide design assets and guidelines to ensure a harmonious design-to-development process. My designs seamlessly translate into HTML/CSS templates, with a focus on responsiveness and consistency, aided by Tailwind CSS.

Website Maintenance and Optimization

In the Website Maintenance and Optimization segment, I demonstrate my commitment to keeping websites running smoothly and efficiently. I offer ongoing maintenance services to ensure clients' websites are up-to-date and secure. Using JavaScript, I optimize website performance for enhanced interactivity and faster load times. Git/GitHub is my preferred choice for version control, allowing for seamless tracking of changes and easy rollback if needed. Plus, I employ Tailwind CSS to streamline styling updates and maintain a cohesive design.
